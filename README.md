# 🛰️ Sentinel AI: QVAC-Powered Offline Financial Hub

**Sentinel AI** is a decentralized, local-first financial dashboard built for the **Solana Colosseum / Frontier Hackathon**. It empowers users in regions with unstable connectivity to manage assets and access AI market intelligence without relying on a constant cloud connection.

---

## 🚀 Key Features

* **🤖 QVAC Offline AI Agent:** A proprietary "Offline Fabric" that allows users to query real-time crypto prices and technical definitions (BTC, SOL, WDK, etc.) entirely on-device.
* **🔑 Tether WDK Integration:** Non-custodial wallet derivation and address generation using industry-standard security protocols.
* **🛰️ Spatial Telemetry:** Integrated GPS locking to verify user location and tie wallet security to physical coordinates.
* **🛡️ Privacy-First Dashboard:** A "Zero-Cloud" approach where sensitive data like telemetry and private key logic stays within the local environment.
* **⚡ Instant UI:** Built with a mobile-optimized splash screen and high-performance dashboard for a seamless user experience.

---

## 🛠️ Technical Stack

* **Languages:** JavaScript (Node.js), HTML5, CSS3
* **Frameworks:** Express.js (Backend Brain)
* **SDKs:** Tether WDK, Google Gemini (Hybrid Cloud Mode)
* **Deployment:** Render

---

## 🏁 Getting Started

1. **Clone the repo:**
   `git clone https://github.com/Sundaycreator20/Sentinel-AI.git`
2. **Install Dependencies:**
   `npm install`
3. **Set Environment Variables:**
   Create a `.env` file and add your `GEMINI_API_KEY`.
4. **Run Locally:**
   `npm start`
